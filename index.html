FRI Campus Tree Map

Interactive map of tree locations on the Forest Research Institute campus.

Species map

This page loads tree data from Airtable and shows markers on the FRI campus using Leaflet and OpenStreetMap tiles.

If you open this page normally, all species will be shown. If you open it with a URL parameter like ?species=Pinus roxburghii, only that species will be shown (assuming the name matches your Airtable data).
